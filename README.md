# 🐼 笹友キャラバン

パンダさんたちの隊商が、元気をなくした町に笹とパンダさんパワーを配って旅する、ドット×放置×ハクスラゲームです。

▶ 遊ぶ：https://fuyutosan.github.io/sasatomo-caravan/

- 放置してもキャラバンは歩き続けて、開くと「るすばん報告」が届きます
- もやもや雲をパンダさんパワーで笑顔にしながら、旅道具を集めて深く旅しよう
- 笹ガチャでなかまのパンダさんをおむかえ。かぶったら覚醒してもっと強く
